## نحوه استفاده

1. فایل `index.html` را در مرورگر باز کنید
2. روی هر مرحله کلیک کنید
3. متن کامل سوره یا ذکر را ببینید
4. با دکمه‌های "قبلی" و "بعدی" بین مراحل حرکت کنید

✅ متن کامل تمام سوره‌ها و اذکار
✅ تصاویر آموزشی
